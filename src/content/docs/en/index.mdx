## Features

<CardGrid stagger>
  <Card title="All CRUD Operations" icon="pencil">

  </Card>
  <Card title="Pagination" icon="document">
    Supports pagination through the 'limit' and 'offset' methods, allowing for
    features like infinite scroll.
  </Card>
  <Card title="Auth with JWT" icon="setting">
    Utilize JWT for access and refresh tokens, and securely test endpoints with

  </Card>

    in the industry.

    execute those queries using your existing data.

  <Card title="Files" icon="add-document">
    You can upload files using `multipart/form-data`.
  </Card>
  <Card title="Postam and Insomia" icon="random">
    We provide files for both Postman and Insomnia, allowing you to easily test
    each endpoint.
  </Card>
  <Card title="AI Images" icon="sun">
    Images are crafted using AI to ensure your projects looks so
    professionalism.
